SQL Data Analysis Project

Overview
This project focuses on analyzing an Sales dataset using SQL.  
The main objective is to explore the dataset, perform basic data cleaning, and extract useful business insights through SQL queries.

Dataset Information
The dataset contains information about:
- Orders
- Products
- Customers
- Payment methods
- Order status
- Revenue
- Referral sources

Objectives
- Explore the dataset using SQL
- Perform basic data cleaning
- Analyze sales and order performance
- Extract business insights from raw data

SQL Concepts Used
The project demonstrates the use of:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- DISTINCT
- UPDATE
- Aggregate Functions

Tools Used
- MySQL
- MySQL Workbench

Key Insights
- Identified top-selling products
- Calculated total sales revenue
- Found the most common payment method
- Explored customer referral trends
- Detected abnormal or invalid records
